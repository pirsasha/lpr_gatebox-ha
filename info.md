## LPR GateBox

Интеграция добавляет в Home Assistant сущности для отображения последнего распознавания номеров из LPR GateBox.

**Сущности:**
- Сенсоры: номер / уверенность / статус / сообщение / время
- Binary sensor: RTSP доступен
- Камера: последний кадр

После установки добавьте интеграцию и укажите Base URL GateBox (например `http://192.168.1.10:8080`).
